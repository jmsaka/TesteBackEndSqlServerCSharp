# Projeto TestBackEndApi

Este projeto tem o objetivo de atender ao teste de Backend fornecido pelo cliente.

## Instruções para utilização

Esse projeto foi desenvolvido utilizando Visual Studio 2019. A princípio, não há dependências de componentes, pois, foram utilizados componentes conhecidos no mercado.

1. Rodar scripts da pasta SQL no banco de Dados Sql Server
2. Trocar o script de conexão do "AppSettings.json"


## Considerações

Esse projeto foi desenvolvido utilizando:

- DDD
- CQRS
- Mediator
- Fluent Validator
- Swagger
- Dapper
- Injeção de Dependência
- Definição de verbos HTTP (conforme requisitos)


## O que "não" foi considerado
- Criação de todos os verbos HTTP
- Camada Cross-Cutting
- Environment
- Swagger - Autorizações, JWT, Claims, Grupos, versões.
- Testes Unitários
